# Writing Good Front Matter

Front matter is the block of YAML at the top of every Markdown file. It's small but important.

## Required Fields

| Field | Purpose |
|-------|---------|
| `slug` | URL identifier |
| `lang` | Language code |
| `title` | Page title |

## Optional Fields

- `date` — publication date, used for sorting
- `author` — displayed on posts
- `tags` — used to group posts

Keep it minimal. Only add fields you actually use.
