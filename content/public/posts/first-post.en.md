# First Post

Welcome to the TeenyTinyCMS blog! This is the first post.

## What is TeenyTinyCMS?

TeenyTinyCMS is a hybrid CMS:

- content body lives in **Markdown** files
- metadata and auth live in a **SQLite or MySQL** database
- output is compiled into **cached PHP** files for maximum performance
