# 404-Seiten richtig handhaben

Eine 404-Seite muss keine peinliche Sackgasse sein. Sie ist eine Chance.

## Was eine gute 404-Seite tut

- Bestätigt, dass die Seite nicht gefunden wurde
- Bietet Navigation zurück zu einer nützlichen Seite
- Behält das Aussehen der Website bei

## In TeenyTinyCMS

Eine `404.de.md`-Seite mit `slug: 404` erstellen. Der Router liefert sie automatisch aus, wenn kein passender Inhalt gefunden wird — mit dem korrekten HTTP-404-Status.
