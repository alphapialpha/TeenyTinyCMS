# Tipps zur Inhaltsorganisation

Eine klare Ordnerstruktur spart später viel Verwirrung.

## Nach Typ trennen

```
content/
  public/
    pages/    ← Über uns, Kontakt, Blog-Index
    posts/    ← einzelne Blog-Einträge
    media/    ← Bilder und Dateien
```

## Namenskonvention

`slug.sprache.md` konsequent verwenden:

```
about.en.md
about.de.md
first-post.en.md
first-post.de.md
```

Der Slug sollte sprachübergreifend gleich sein, damit der Sprachwechsler sie verknüpfen kann.
