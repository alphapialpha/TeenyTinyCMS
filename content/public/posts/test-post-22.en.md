# Content Organisation Tips

A clear folder structure saves a lot of confusion later.

## Separate by Type

```
content/
  public/
    pages/    ← about, contact, blog index
    posts/    ← individual blog entries
    media/    ← images and files
```

## Naming Convention

Use `slug.lang.md` consistently:

```
about.en.md
about.de.md
first-post.en.md
first-post.de.md
```

The slug should match across languages so the language switcher can link them.
