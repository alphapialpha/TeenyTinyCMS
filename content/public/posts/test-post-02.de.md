# Einstieg in Markdown

Markdown ist eine einfache Auszeichnungssprache, mit der man formatierten Text als reinen Text schreibt.

## Warum Markdown?

- Leicht lesbar und schreibbar
- Wird zu sauberem HTML gerendert
- Überall unterstützt

Ausprobieren lohnt sich — die meiste Syntax kennt man schon intuitiv.
